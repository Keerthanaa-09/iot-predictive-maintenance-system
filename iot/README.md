## IoT Firmware

This folder contains firmware code for **sensor data acquisition** using **ESP8266**.

- **esp8266_temperature.ino** – Reads temperature data from the **DS18B20** sensor  
- **esp8266_vibration.ino** – Captures vibration data from the **ADXL345** sensor  
- Sensor data is transmitted to the cloud via **Wi-Fi** for monitoring and analysis
